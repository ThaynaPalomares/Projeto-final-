# Processo de Desenvolvimento

## Metodologia Escolhida
Foi adotado o método Kanban.

## Justificativa
O Kanban permite organizar o fluxo de trabalho de forma simples, sendo ideal para projetos com escopo definido e desenvolvimento individual.

## Estrutura do Board
- Backlog
- A Fazer
- Em Progresso
- Concluído

## Funcionalidades
1. Busca de perguntas
2. Categorização
3. Votação
4. Perfil de usuário
5. Notificações

## Priorização
A busca foi priorizada por melhorar diretamente a experiência do usuário. As demais funcionalidades foram organizadas por impacto e complexidade.
