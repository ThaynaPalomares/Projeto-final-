# Processo de Desenvolvimento

## Metodologia Escolhida
Foi adotado o método Kanban para organização do fluxo de trabalho.

## Justificativa
O Kanban permite visualizar claramente as etapas do desenvolvimento, facilitando o acompanhamento das tarefas e a priorização contínua. É adequado para projetos com escopo definido e desenvolvimento individual.

## Estrutura do Board
O quadro Kanban foi estruturado nas seguintes colunas:
- Backlog
- A Fazer
- Em Progresso
- Concluído

## Board do Projeto
O gerenciamento das tarefas foi realizado através do GitHub Projects:
https://github.com/users/ThaynaPalomares/projects/3

## Funcionalidades
As seguintes funcionalidades foram definidas para o sistema:
1. Busca de perguntas por palavra-chave
2. Categorização de perguntas
3. Sistema de votação em perguntas
4. Perfil de usuário com histórico
5. Notificação de novas respostas

## Priorização
A funcionalidade de busca foi priorizada por impactar diretamente a usabilidade do sistema. As demais foram organizadas considerando valor para o usuário e complexidade de implementação.
