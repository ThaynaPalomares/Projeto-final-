# Padrões de Projeto Existentes – ESM Forum

## 1. Component-Based Design
O frontend em React adota uma abordagem baseada em componentes, com separação em elementos reutilizáveis e páginas específicas.

## 2. Router como estrutura de navegação
O uso do React Router organiza a navegação entre páginas e centraliza o fluxo da aplicação.

## 3. Cliente-Servidor
O sistema adota uma estrutura cliente-servidor, em que o frontend consome dados de uma API backend por meio de requisições HTTP.

## Conclusão
Os padrões identificados não estão formalizados como padrões GoF clássicos em todos os casos, mas já demonstram preocupação com modularidade e separação estrutural.
