# Histórias de Usuário

## História 1: Busca de Perguntas

**Como** usuário do fórum,  
**Eu quero** buscar perguntas por palavras-chave,  
**Para** encontrar rapidamente conteúdos relevantes.

**Critérios de Aceitação:**
- [ ] O sistema deve permitir inserir palavras-chave em um campo de busca
- [ ] O sistema deve retornar uma lista de perguntas que contenham o termo pesquisado
- [ ] A busca deve considerar o título e o conteúdo da pergunta
- [ ] O sistema deve exibir mensagem quando não houver resultados

---

## História 2: Categorização de Perguntas

**Como** usuário do fórum,  
**Eu quero** categorizar perguntas com tags,  
**Para** organizar melhor os conteúdos.

**Critérios de Aceitação:**
- [ ] O usuário deve poder selecionar ou informar tags ao criar uma pergunta
- [ ] O sistema deve exibir as tags associadas a cada pergunta
- [ ] O sistema deve permitir filtrar perguntas por categoria
- [ ] As tags devem seguir um padrão de organização

---

## História 3: Sistema de Votação

**Como** usuário do fórum,  
**Eu quero** votar em perguntas com upvote ou downvote,  
**Para** destacar os conteúdos mais relevantes para a comunidade.

**Critérios de Aceitação:**
- [ ] Cada pergunta deve exibir opção de voto positivo e negativo
- [ ] O sistema deve atualizar a contagem de votos após a ação do usuário
- [ ] O usuário não deve poder votar múltiplas vezes da mesma forma na mesma pergunta
- [ ] O usuário deve poder alterar seu voto

---

## Priorização

1. Busca de Perguntas  
2. Categorização de Perguntas  
3. Sistema de Votação  

## Justificativa da Priorização

A busca foi priorizada por melhorar diretamente a navegação e a experiência do usuário.  
A categorização vem em seguida por facilitar a organização do conteúdo.  
A votação foi posicionada em terceiro lugar por agregar relevância ao conteúdo após a estrutura de busca e organização já estar definida.
