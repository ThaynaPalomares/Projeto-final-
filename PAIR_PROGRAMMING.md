# Pair Programming

## Estratégia
A prática pode ser simulada por meio de revisões frequentes do código.

## Papéis
- Driver: escreve o código
- Navigator: revisa e orienta

## Rotação
Alternância de papéis para melhorar a qualidade do código.

## Ferramentas
- VS Code Live Share
- GitHub
- Discord ou Meet

## Benefícios
- Redução de erros
- Melhor qualidade
- Decisões mais claras
